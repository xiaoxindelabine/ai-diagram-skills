# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-07-13

- Generalized `nn-diagram` from a local project-specific skill into a reusable AI diagram skill.
- Added generic templates for neural networks, RAG systems, agent workflows, multimodal models, and training/evaluation pipelines.
- Converted OutputFusion and related project-specific material into an optional case library.
- Added bilingual open-source documentation and MIT license.
- Added lightweight validation for frontmatter, local references, and path leakage.
